Usage suggestions
- Use `--brand` for headings and key UI elements.
- Use `--accent` and `--orange` for CTA buttons and highlights.
- Reserve `--red`/`--bright-red` for alerts or important badges.
- Use `--navy` / `--navy-2` for deep backgrounds or nav bars.

Notes
- These values are approximations taken from hex byte blocks inside the EPS file; for exact color matching, convert the EPS to SVG/PNG and sample colors in a graphics editor.
